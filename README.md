## How to Run Locally

Note: Ensure Node.js and npm are installed on your system

```bash
git clone https://github.com/krupesh-desai/FinTrack-FullStack-Finance-Platform.git
cd FinTrack-FullStack-Finance-Platform
```

## Backend

```bash
cd backend
npm install
npm start
```

## Frontend

```bash
cd frontend
npm install
npm start
```
